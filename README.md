# PulsePoint
PulsePoint🧭 is an interactive web app for sharing and discovering real-time events, issues, and points of interest on a map.🗺️

## Installation and compilation🖥️
### Frontend🖼️
```bash
cd frontend
```
```bash
npm install
```
```bash
npm start
```

### Backend⚙️
```bash
cd backend
```
```bash
.\mvnw clean spring-boot:run
```
